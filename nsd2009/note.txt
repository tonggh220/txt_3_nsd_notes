ssh 黑名单： 仅不允许使用名单列表里的用户访问ssh服务
DenyUsers    nb@192.168.4.200   wk

ssh 白名单：仅允许使用白名单列表里的用户访问ssh服务
AllowUsers       wk

ssh服务验证客户连接的方式 支持2种
			1 密码 （连接用户登录系统的密码）
			2 密钥对 （公钥文件   私钥文件）
                                加密         解密
                                锁头         钥匙

#####安全_day04   iptables软件提供的防火墙服务
                              firewalld软件提供的防火墙服务）
		  iptables介绍
		  
				安装提供iptables服务软件包并启动服务
				
				
				数据在网络传输时，要被封装成IP包   
				
				我爱你   协议（端口号   传输数据使用的协议   源ip地址 目标地址  Mac）
				
		  iptables组成 （重点）
		  4张表  （服务的功能分类）
		         包过滤表 filter 
                 地址转换表 nat
                 包标记表  mangle
                 状态跟踪表  raw
				 
		  
		  5种链   (ip包的传输方向)  以防火墙服务器为参照物
		  
		  INPUT  进入防火墙的数据包
		  OUTPUT  从防火墙本机出去的数据包
		  FORWARD   经过防火墙服务器的数据包
		  PREROUTING  路由前处理   
		  POSTROUTING 路由后处理
		  
iptables -t  raw  -L
iptables -t  filter  -L
iptables -t  mangle  -L
iptables -t  nat  -L
